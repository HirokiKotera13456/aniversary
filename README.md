# 記念日カウンター (Next.js)

2025年6月23日からの交際日数を記録し、冬風・夏風の2つの世界観をワンタップで切り替えられる Next.js (pages router) アプリです。

## セットアップ

> ネットワークに制約がある環境では `npm install` に失敗する場合があります。必要に応じて社内レジストリなどをご利用ください。

```bash
npm install
npm run dev
```

## 主な特徴

- pages router の `pages/index.tsx` にロジックを集約
- リアルタイムで日数/時間/分/秒をカウント
- 冬と夏のスタイルを CSS で切り替え
- マイルストーンカードやタイムラインで今後の予定を可視化
